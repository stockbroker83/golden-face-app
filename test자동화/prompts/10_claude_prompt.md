# Claude 요청 템플릿

아래 조건으로 **youtube-analyzer 코드 변경 초안**을 만들어줘.

## 작업 내용
- 대상 프로젝트: `C:\Users\csu\OneDrive\Desktop\깃허브코드\youtube-analyzer`
- [여기에 작업 내용을 작성]

## 파이프라인 범위 (고정)
- 대본 입력
- 문장 분할
- 이미지 프롬프트 생성
- 이미지 생성
- 자막 생성
- MP4 렌더링

## 제외 범위 (중요)
- TTS 관련 작업은 이번 범위에서 제외
- elevenlabs 연동/키 설정/음성 합성 코드 생성 금지

## 출력 형식
1. 변경 계획 (3~5줄)
2. 수정할 파일 목록
3. 핵심 코드 변경안
4. 검증 명령어

## 제약
- 기존 스타일 유지
- 불필요한 파일 생성 금지
- 민감정보 하드코딩 금지
